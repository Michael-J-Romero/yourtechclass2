# YourTechClass

A single-page website built with **Next.js**, **React**, and **Material UI** promoting afterschool coding groups at local libraries in Los Angeles.

## Features

- **Hero section** – Bold call-to-action with links to locations and the contact form
- **Programs overview** – Cards highlighting Afterschool Groups, One-on-One Lessons, and Project Showcases
- **Locations section** – Six LA library locations with addresses, session days/times, and age ranges
- **One-on-One Lessons section** – Details and benefits of personalized private instruction
- **Contact form** – Name, email, phone, and message fields with client-side validation

## Tech Stack

- [Next.js 16](https://nextjs.org/) (App Router)
- [React 19](https://react.dev/)
- [Material UI 7](https://mui.com/)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
npm run start
```
